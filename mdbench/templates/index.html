Molecular Dynamics Benchmarks

Benchmarking is essential for finding the optimal job simulation and submission parameters. New users might be unaware of the hardware- and software-specific details required to run efficient simulations on Compute Canada HPC clusters. A poor choice of parameters can increase the simulation speed only by a narrow margin or even degrade it leading to a waste of computing resources. Therefore, it is in the interest of every user to optimize their usage of hardware resources and, at the same time, keep their simulation time to a minimum.

This database provides researchers with detailed information about the performance and efficiency of MD simulations on heterogeneous Compute Canada clusters. Currently, it contains benchmarks for three major MD software packages: AMBER, GROMACS, and NAMD. This information should allow researchers to estimate the performance they can expect for similar simulations. All entries include example submission scripts and simulation input parameters to help new users to get started with MD simulations and optimize submission of their simulations.
